Upload your network data

After navigating to the tab 'User Input Data', you can specify the network file type (figure,panel A):

  • Edge list
  • Adjacency matrix
  • Cytoscape sif file

The Cytoscape sif file is the standard output file one can generate in the Cytoscape program and is intended to allow you to visualize your network you've already build in Cytoscape.
The adjacency matrix (of dimension MxM) should be of the standard format thus an matrix containing zeros if there is no connection and ones if there is a connection between nodes (figure, panel B).
The edge list should be a two column file containing the source node in column 1 and the target node in column 2. (figure, panel C)

Upload the experimental data

The second step is uploading the experimental data from the dose/time/dose-over-time series. These data values are used to color the nodes in the network, for instance to show the log intensity changes of the genes in a gene regulatory network. Each of the different file, whether it is a dose series or time series has a comparable data file structure. The rows in the file correspond to the different nodes in the network and therefore should have the same name. The columns in the file correspond to the dose, time point or Dose X at time point Y (figure, panel D).
As a last option, the user can upload an interaction file, which is a file that will specify which interactions are present at different time points.

Welcome at the DynOVis visualization tool for network analysis

DynOVis is designed to capture the dynamic nature of biological data due to different perturbations, such as time or dose series experiments.
Integrative visualization can be used to gain more information out of the networks graphs and thereby increase our understanding of the biological data.

Here, we have integrated a dynamic visualization approach combined with integrated databases, to add biological context to the network. This includes information about related pathways, associated diseases and general gene information.

Where are my images saved to?

By default, your images will be downloaded to the standard folder specified by your browser.
In general, this is 'Downloads' folder in your documents directory.

How to use the tutorial data

DynOVis has two different tutorial data sets that can be viewed in the tool: i) NFkB pathway exposed to acetaminophen (NFkB data set) and ii) biological network of Fibronectin 1 perturbation after primary human hepatocytes have been exposed to Valproic acid.

To upload the data, just check one of the two boxed (NFkB exposure or VPA exposure) and press the button ‘Upload data’. The second step will be to select the directionality of the network: directed or undirected. For the Tutorial data sets please select ‘directed’. Finally, you can select if you want to map the genes in the network to the internal database of DynOVis. For the NFkB exposure pathway you can select 'map nodes to database' and select 'Entrez ID'. For the tutorial data set of VPA exposure you should select 'gene Name'. To start the visualization press ‘Build network’.

A short description of the main functions:
Threshold on node degree: If you click on the menu bar, the panel will drop down and three options sliders become visible to select a threshold on the degree for which a node becomes hidden. The drop menu can be closed again by clicking on 'Threshold on node degree'
Show Expression data: This drop down menu contains the different animation options of DynOVis: i) color nodes with expression ii) Edge interaction over time and iii) color node + edge interaction. The animation controls are used to play the animation ('play button'),pause/resume the animation ('pause'button) and to go frame-by-frame through the network ('forward/backward' button).
For the NFkB data set choose the option: color nodes with expression
For the VPA exposure set choose the option: color nodes + edge interaction

Start analysis of subselection: Before you can run this option, you first have to double click on a node in the network to highlight the node and its first neighbors. After this step, you can press 'start analysis subselection' which will create a 2D line graph of the expression changes over time or dose.

A complete tutorial which explains all the functions of the tool in more detail can be found here.
